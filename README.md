# Лабораториска 1: Корисничка Автентикација со Flask и PostgreSQL
## Преглед
Веб-апликација за безбедна корисничка автентикација со имплементација на основни концепти од информациската безбедност. Апликацијата овозможува регистрација, пријава (логирање), и одјава (логаут) на корисници, со фокус на безбедносна заштита од XSS, SQL Injection и Session Hijacking напади.

### Цел и Задачи

 ✅ **Регистрација на нови корисници** -
Валидација на email и лозинка со проверка на сложеност.

✅ **Пријава и сесиски менаџмент** -
Flask-Login за управување со сесии и ограничување на приватни страници.

✅ **Хеширање на лозинки со salt** -
SHA-256 со уникатен salt генериран за секој корисник.

✅ **XSS заштита** - 
Санитизирање на кориснички input - замена на < и > со HTML ентитети.

✅ **SQL Injection заштита** - 
SQLAlchemy со parameterized queries - спречување на SQL манипулации.

✅ **Session Cookie заштита** - 
HttpOnly, Secure, и SameSite флагови за заштита од Session Hijacking.

✅ **PostgreSQL база** - 
Складирање на кориснички податоци со уникатни username и email.

### Користени Технологии
- Backend: Python 3.x, Flask

- Датабаза: PostgreSQL 15.x

- ORM: SQLAlchemy

- Управување со сесии: Flask-Login

- Хеширање: hashlib (SHA-256) со salt

- Заштита: XSS санитизирање, CSRF заштита

- Frontend: HTML5, Jinja2 templates
